Metodos de la API REST

Mostrar Editoriales

GET https://localhost:4200/editorials

Muestra el id, nombre, cantidad de empleados, fundador, el jefe actual, la fecha de creacion y el pais de origen de cada editorial.

Mostrar datos de 1 superheroe

GET https://localhost:4200/editorials/:id/

Muestra los datos de solo ese editorial

Mostrar Superheroes

GET https://localhost:4200/superheroes

Muestra el id, nombre y la primera aparicion de cada superheroe. Tambien muestra el id de la Editorial a la cual pertenece cada superheroe.

Mostrar datos de 1 superheroe

GET https://localhost:4200/superheroes/:nombre/

Muestra los datos de solo ese superheroe.

Mostrar los superheroes de 1 editorial

GET https://localhost:4200/superheroes/editorial/:id/

Muestra la informacion de todos los superheroes de la editorial escogida.
